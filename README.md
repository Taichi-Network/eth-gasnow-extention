# gasnow-extention
GasNow extension for browser

## Prerequisites
    yarn
	nodejs

## Install

	$ yarn

## Development

    ### chrome
    yarn start:chrome

Change some code.

## Build
In the new terminal.

    yarn build:all

## Environment

The build tool also defines a variable named `process.env.NODE_ENV` in your scripts.
